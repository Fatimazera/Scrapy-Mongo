Scrapy + MongoDB Book Scraper

A web scraping project built using Scrapy that extracts data from [Books to Scrape](https://books.toscrape.com/) and stores it in MongoDB Atlas

Features:

- Scrapes  data from multiple categories
- Stores data in MongoDB collections dynamically
- Uses environment variables for secure database connection

Tech Stack:

- Python
- Scrapy
- MongoDB (Atlas)
- PyMongo
